    1. 创建hexo

    创建hexo

    1. 创建仓库

    在 github 新建一个仓库，仓库名必须为 yzhengwang.github.io 格式，其中 yzhengwang 是你 github 的名称。

    1. 安装hexo

    npm install -g hexo

    1. 初始化项目

    hexo init，会自动构建一个 hexo 项目，执行 hexo s，浏览器访问 localhost:4000 就可以看到效果！

    1. 部署到github

    在项目根目录下找到 _congif.yml，找到 deploy 字段并填写完整。

    deploy:  
  type: git  
  repo: <你的仓库地址> # git@github.com:yzhengwang/yzhengwang.github.io.git  
  branch: master

    我们需要额外的一个工具来帮助我们推到仓库上，hexo-deployer-git。

    $ npm install hexo-deployer-git --save  
$ hexo clean  
$ hexo deploy

    浏览器访问：https://yzhengwang.github.io/ 即可看到效果。

    1. 创建新文章
    # [layout] 为布局，可选项为 `post`、`page`、`draft`，这将决定文章所在文件路径。
# <title> 为文章标题
# 如 hexo new post 
$ hexo new [layout] <title>
    1. 更换主题皮肤
    • 下载到themes文件夹下
      git clone https://github.com/KevinOfNeu/hexo-theme-tttt tttt
    • 修改 _config.yml 配置
      theme: tttt
    1. 部署优化

    每次都要执行 hexo clean 和 hexo deploy，不如写个新的脚本

    // package.json  
"dev": "hexo s",  
"build": "hexo clean & hexo deploy"

    部署命令

    $ npm run build
